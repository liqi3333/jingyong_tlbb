# 第八章：虎啸与龙吟——《天龙八部》的权力博弈与救赎之路

---

### 一、金庸笔下的棋局隐喻——智慧、内力与命运的较量

金庸在武侠小说中多次运用"棋局"作为情节的核心意象。《天龙八部》中黄眉僧与段延庆的棋局对决，是其中最精彩的一例。棋局作为中国传统文化中的"雅趣"之一，在金庸笔下被赋予了全新的叙事功能——它不仅是一种智力的较量，也是一种武功的较量，更是一种命运的隐喻。

在中国传统文化中，棋被视为"文人四艺"（琴棋书画）之一，与琴、书、画并列，代表了文人的修养和品位。金庸将"棋"引入武侠小说，不仅增加了小说的文化底蕴，也创造了一种全新的"比武"方式——不是刀光剑影的血腥厮杀，而是运筹帷幄的智力较量。这种"文斗"的形式，在武侠小说中具有开创新意义——它表明，在武侠世界中，"智力"和"武力"同样重要，甚至在某种意义上，"智力"比"武力"更加重要。

金庸对"棋"的运用，还可以追溯到中国传统文化中"棋如人生"的思想。古人说"世事如棋"，意思是人生的变化就像棋局一样，充满变数和不确定性。金庸通过段延庆与黄眉僧的棋局，将这种"棋如人生"的哲学具象化了——两人的每一步棋，都对应着他们在现实中的每一步抉择；棋局的输赢，直接影响着现实的命运。这种"以棋喻人、以局说事"的写法，是金庸小说中最具文化意蕴的部分之一。

**棋局作为"文斗"的极致**

在武侠世界中，高手对决通常以拳脚兵器展开。然而黄眉僧与段延庆的对决，选择了另一种方式——下棋。这盘棋不仅仅是棋力的较量，更是内力的较量——黄眉僧用铁槌在青石上刻下棋路，段延庆用铁杖在石上捺出印记，每一着都包含着深厚的内力。

这种"以棋比武"的写法，在金庸的创作中并非首次出现。在《倚天屠龙记》中，昆仑三圣何足道以棋艺与少林寺的高手较量；在《笑傲江湖》中，黑白子的棋局也成为日月神教武功的一部分。但黄眉僧与段延庆的棋局对决，是金庸笔下最为精彩、最为复杂的"棋战"——它不仅融合了棋力和内力，还涉及了心理战、耐力和智慧的多重较量。

黄眉僧以铁槌为棋子，在坚硬的青石上刻出棋盘和棋路。每刻一子，都需要灌注深厚的内力。他用槌在石上一敲，石屑纷飞，一个棋子就刻出来了。段延庆则用铁杖在石上捺出印记，虽然无声无息，但力道更加深沉。两人就在这片青石上，展开了一场无声的厮杀。

金庸将"棋"与"武"融合在一起，创造了一种独特的"文斗"形式。文斗比武斗更难写，因为它需要作者同时精通棋理和武学。金庸以"棋"写"武"，以"局"写"人"，使这场对决既是一场智力之争，也是一场生死之战。通过这场棋局，金庸展示了中国传统文人的"雅趣"——即使是生死对决，也可以以一种"文明"的方式进行。

**旁观者清的段誉**

段誉自幼善弈，棋力本就不低于黄眉僧。他被困石屋之中，隔墙观棋，忍不住出声指点。他的每一步棋都直指要害，令黄眉僧茅塞顿开。

这里有一个有趣的反讽——段誉被囚禁在石室中，身中春药，性命垂危，却能在棋局上保持清醒的判断力。身体的困厄与心智的清明形成了鲜明的对比。这也体现了段誉的性格核心——无论身处何种困境，他都能保持头脑的清醒。

段誉的"隔墙指点"，在叙事上具有多重功能。首先，它展现了段誉的智力——他虽然是"三个主角中最不会武功的一个"，但他是"三个主角中最聪明的一个"。他的棋力、他的学识、他的修养，都是其他两个主角无法比拟的。其次，它为段誉提供了"参与"这场对决的方式——他虽然被囚禁在石室中，但他通过"声音"参与到了外界的战斗中，这种"间接参与"在叙事上增加了段誉的存在感。第三，它为后续的情节发展埋下了伏笔——段誉的"指点"不仅帮助了黄眉僧，也引起了段延庆对他的注意。

**棋局的象征意义**

黄眉僧与段延庆的棋局，不仅是一场个人的对决，更象征着大理国正邪两种力量的较量。黄眉僧代表了"正则"——他是保定帝请来的帮手，代表了正统的力量；段延庆代表了"邪道"——他是四大恶人之首，代表了邪恶的力量。两人的棋局，实际上是大理国命运的缩影。

棋局的"僵局"结局，也具有深刻的象征意义。正邪双方势均力敌，谁也无法消灭谁——这种"平衡"在某种意义上就是那个时代的真实写照。大理国的命运，正如同这盘棋——在正邪两种力量的拉锯中，既不能完全偏向正，也不能完全偏向邪，只能在这两种力量之间寻找一种微妙的平衡。

**棋局与易经的联系**

段誉在指点棋局时，引用了《易经》的卦象——"观我生，进退。艮其背，不获其人；行其庭，不见其人。鼎耳革，其行塞。"这些卦辞表面上是说棋局的变化，实际上也是在说人的命运——"观我生"是对自我的审视，"进退"是在困境中的选择，"艮其背"是面对危险时的姿态。金庸通过段誉的口，将棋局、易理、命运三者巧妙地联系在一起，使这场棋局对决具有了超越"比武"本身的哲学意蕴。

### 二、保定帝段正明——仁君形象与家国情怀

段正明（保定帝）在金庸笔下是一个极具魅力的君主形象。他仁慈宽厚、智谋深沉，是金庸作品中难得的"好皇帝"。在武侠小说中，"皇帝"通常被塑造成昏庸无能或残暴不仁的形象，但保定帝却是一个"仁君"的典范。

**兄弟情深**

保定帝与段正淳兄弟情深。段誉被囚禁后，保定帝比段正淳更为焦虑。他不仅以皇帝的身份下令营救，更以伯父的身份亲自主持应对。这种兄弟之情，超越了君臣之别，使保定帝的形象更加人性化。

保定帝对段誉的关心，体现了中国传统文化中"家国一体"的理念。在保定帝看来，段誉不仅是他的侄子，也是大理国的储君——保护段誉的安全，既是"家事"也是"国事"。这种"家国不分"的治理方式，在当时的政治环境中是合理的——君主既是国家元首，也是家族家长，他的"家事"往往就是"国事"。

**以退为进的智慧**

当保定帝得知段延庆的真实身份后，他的第一个反应是让位——"皇位本来是延庆太子的。当日只因找他不着，上明帝这才接位，后来又传位给我。延庆太子既然复出，我这皇位便该当还他。"

这一表态，既是真诚的——他不贪恋权力，愿意为了侄子的安全而放弃皇位；也是高明的——他以退为进，让段延庆无法轻易下手。段正淳和高升泰等人极力劝阻，保定帝最终放弃了让位的念头，但这一表态已经展现了他作为君主的胸怀。

保定帝的"让位"表态，在政治策略上是一种非常高明的做法。它达到了三个目的：第一，它表明保定帝是"重义轻利"的人——他视亲情高于权力；第二，它向段延庆传递了一个信息——"我不是你的敌人，我愿意把属于你的东西还给你"；第三，它使段正淳和大理群臣更加坚定地支持保定帝——因为保定帝是一个值得拥护的"好皇帝"。

**家国一体的治理理念**

保定帝虽然在听到万劫谷的名字之前不知道这个地点，但他对大理的治理却是尽心尽力的。他接受黄眉僧的请求免除了盐税，展现了仁君的气度。他在处理段誉被囚一事上，既展现了温情，又保持了理智，是一个真正的"明君"形象。

保定帝的治理理念，体现了儒家"仁政"的思想。他减免赋税是因为"百姓疾苦，感同身受"，他营救段誉是因为"骨肉之情，不可弃也"，他冷静决策是因为"一国之君，岂可轻举妄动"。这种"仁"与"智"的结合，使保定帝成为一个理想化的君主形象。

在《天龙八部》中，保定帝的出现还有一个重要的叙事功能：他为段誉提供了一种"对比"——段誉将来也要继承皇位，成为大理国的一国之君。保定帝的仁厚和智慧，是段誉学习的榜样。虽然段誉最终并没有"学习"保定帝的为君之道——他选择了按照自己的方式治理国家——但保定帝的存在，使读者对段誉未来的"皇帝生涯"有了一个基本的期待。

### 三、华赫艮的"盗墓"出身与大理三公的智慧

金庸善于在小说中安排"奇人异士"，华赫艮就是其中之一。他的出现，为这个沉重的故事增添了一抹幽默和趣味的色彩。

**从盗墓者到司徒**

华赫艮本名华阿根，出身贫贱，年轻时干的是盗墓掘坟的勾当。他善于从远处挖掘地道通入坟墓，盗取宝物。一次偶然的机会，他在棺木中得到了一本武功秘诀，依法修习，练成了一身卓绝的外门功夫。此后他舍弃了盗墓的营生，辅佐保定帝，累立奇功，终于升到司徒之职。

这一背景设定，既有趣又有深意——一个曾经的盗墓贼，后来成为了一国之司徒。金庸以这一人物的转变，传达了"放下屠刀，立地成佛"的理念。华赫艮的故事告诉我们：无论一个人过去的身份如何，只要他愿意改变，就有机会获得新生。

华赫艮的"盗墓"出身，在中国传统文化中具有特殊的象征意义。盗墓者通常被视为最低贱、最不道德的人——他们挖掘死者的坟墓，盗取死者的财物，这种行为严重违背了中国传统文化中"死者为大"的伦理规范。然而金庸却让这样一个"道德上有污点"的人成为了大理国的高官——这种设定本身就是对传统道德观念的一种挑战。金庸似乎在说：一个人的价值不应该由他的过去决定，而应该由他的现在和未来决定。

**挖地道的妙计**

华赫艮虽然身居高位，但一听说挖地道，当场就来了精神——他于盗墓一事有天生嗜好，二十年来虽然再不干此营生，偶尔想起，仍是禁止不住手痒。范骅提议挖地道救段誉，华赫艮一拍大腿，叫道："妙极，妙极！"

范骅笑道："大哥且慢欢喜，这中间着实有些难处。四大恶人都在万劫谷中，钟万仇夫妇和修罗刀也均是极厉害的人物，要避过他们耳目委实不易。"

华赫艮沉吟半晌，说道："地道当从石屋之后通过去，避开延庆太子的所在。"范骅道："镇南世子时时刻刻都有危险，咱们挖掘地道，只怕工程不小，可来得及么？"华赫艮道："咱哥儿三人一起干，委曲你们两位，跟我学一学做盗墓的小贼。"巴天石笑道："既然位居大理国三公，这盗墓掘坟的勾当，自是义不容辞。"三人一齐拊掌大笑。

这一幕写得极为生动——三位高官放下身段，去干"盗墓"的勾当，却干得兴高采烈。金庸以幽默的笔调，弱化了营救行动的紧张感，也展现了大理智囊团的智慧和团结。

华赫艮的"地道计划"，在叙事上体现了"以巧取胜"的思想。面对四大恶人的围困，硬碰硬是最愚蠢的选择——因为段延庆的武功深不可测，强行进攻必然造成重大伤亡。而挖地道救人，虽然"不光彩"——堂堂一国司徒居然要干盗墓的勾当——但却是最有效、最安全的方案。这种"不择手段"（在道德允许的范围内）的务实作风，是金庸笔下"智囊"人物的共同特点。

### 四、段延庆——"四大恶人"之首的悲剧根源

段延庆是"四大恶人"之首，外号"恶贯满盈"。他的悲剧根源，在于他被命运彻底抛弃。在《天龙八部》的"恶人谱"中，段延庆是最复杂、最令人同情的一个。

**从太子到恶人**

段延庆原本是大理国的太子（延庆太子），是皇位的合法继承人。然而在上明帝时期，朝中发生政变，延庆太子不知去向，人人都以为他被杀了。上明帝这才接位，后来又传位给段正明。

段延庆在外流亡多年，历尽苦难，身体残废，面目全非。他从一个尊贵的太子，沦落为江湖上最凶残的恶人。这种身份的落差，使他对段氏家族充满了怨恨。他要夺回皇位，不仅是为了权力，更是为了报复那些"夺走"他一切的人。

段延庆的悲剧，在某种意义上与乔峰的悲剧形成了对称。两人都是命运的受害者——乔峰被从"汉人英雄"变成了"契丹恶人"，段延庆被从"大理太子"变成了"四大恶人"。两人的区别在于：乔峰在面对命运打击时选择了"抗争"，而段延庆选择了"堕落"。乔峰即使被误解、被抛弃，也从没有放弃做人的尊严和底线；段延庆则在痛苦的折磨中失去了对人性的信仰，变成了一个以"恶"为生存方式的人。

**"恶贯满盈"的哲学**

段延庆的"恶"，不是天生的，而是被逼出来的。他曾经是太子，拥有尊贵的身份和光明的未来。然而政变摧毁了他的一切——他失去了地位、失去了家人、失去了健全的身体。在这种极端的痛苦中，他选择了以"恶"作为自己的生存方式。

然而段延庆的"恶"与南海鳄神、叶二娘、云中鹤的"恶"不同。南海鳄神的恶是凶暴——他杀人如麻，以暴力为乐；叶二娘的恶是残忍——她专门残害婴儿，以他人的痛苦为乐；云中鹤的恶是淫邪——他好色无度，以蹂躏女性为乐。而段延庆的恶，更像是一种悲哀——他以恶的姿态出现，却在内心深处保留着对"正"的认知。

段延庆的"恶贯满盈"这个外号，本身就具有双重的含义。一方面，"恶贯满盈"是指他的恶行已经到了"装满"（盈）的程度——他是一个彻头彻尾的恶人。另一方面，"恶贯满盈"也可以理解为"恶"已经"贯穿"（贯）了他的"满盈"（全部）——他的全部身心都被"恶"所占据。但段延庆的"恶"，与其说是他主动选择的，不如说是被命运强行灌输的。

**段延庆与段誉的宿命联系**

段延庆是段正淳的妻子刀白凤"报复"的对象——刀白凤因为段正淳的风流，愤而与段延庆发生了一夜之情，生下了段誉。这意味着段誉真正的生父不是段正淳，而是段延庆。这一真相虽然要到后面才揭晓，但段延庆在这一章中的出场，已经为这一惊天秘密埋下了伏笔。

段延庆与段誉的"父子关系"，是《天龙八部》中最令人震惊的设定之一。它让读者重新审视段誉的"身世"——段誉一直以为自己是段正淳的儿子，段正淳也一直以为段誉是自己的儿子。但真相是：段誉是段延庆的儿子，而段延庆是段氏家族的"敌人"。这种"敌人成了父亲"的戏剧性反转，在叙事上具有强烈冲击力。

金庸通过段延庆与段誉的关系，再次诠释了"命运无常"的主题。段延庆一生都想夺回皇位，但他不知道的是，他的儿子段誉——那个由他仇人的儿子"抚养"长大的少年——才是真正继承了段氏血脉的人。而当段延庆最终得知真相时，他所有的执念——对皇位的追求、对段氏家族的仇恨——都在这一刻消解了。他放弃了争夺皇位，将自己的"接力棒"传给了段誉。

### 五、"虎啸龙吟"的意象与权力角逐

"虎啸龙吟"作为第八章的标题，具有多重象征意义。在中国传统文化中，"虎"和"龙"不仅代表着强大和尊贵，也代表着"阳"与"刚"的力量。金庸以"虎啸龙吟"为标题，预示着本章中两种强大力量的激烈碰撞。

从《天龙八部》前八回的回目来看，金庸在回目的命名上有一个有趣的现象——他会根据故事的氛围选择相应的意象。前七回的回目（"青衫磊落险峰行""玉壁月华明""马疾香幽""崖高人远""微步縠纹生""谁家子弟谁家院""无计悔多情"）更多地偏向于"情"和"景"的描写，营造了一种"诗化"的氛围。而第八章"虎啸龙吟"则是一个转折——它标志着故事从"柔情"转向了"刚烈"，从"个人"转向了"群体"，从"情感纠葛"转向了"权力斗争"。

**虎与龙的意象**

在中国传统文化中，虎和龙分别代表两种强大的力量。虎是百兽之王，代表了威武和力量；龙是神兽之首，代表了尊贵和权威。虎啸龙吟，意味着两种强大力量的相遇和对抗。

在第八章中，"虎"指段延庆——他被命运抛弃，却拥有惊人的武功和决绝的意志，如同猛虎下山；"龙"指保定帝——他是一国之君，代表了正统和秩序，如同天龙在渊。两人的对抗，是"虎"与"龙"的较量。

值得注意的是，金庸在小说中对"虎"和"龙"的使用，并非简单地"龙=正、虎=邪"。段延庆虽然是"虎"，但他并不是彻头彻尾的恶人——他有他的悲剧原因和值得同情之处。保定帝虽然是"龙"，但读者在阅读中也会发现他的局限性——他虽然是仁君，但他面对段延庆时也有束手无策的时候。金庸的这种"不简单化"的处理，使"虎啸龙吟"的意象更加丰富、更加深刻。

**权力角逐的暗线**

段延庆之所以出现在万劫谷，表面上是受钟万仇之邀来对付段正淳，实际上是为夺回皇位而来。他以段誉为筹码，逼迫保定帝让位。这种权力角逐，是第八章的暗线——表面是江湖恩怨，实质是皇位之争。

段延庆的"权力"策略，体现了他作为政治家的手腕。他虽然没有直接向保定帝宣战，但通过绑架段誉——大理国的储君——他向保定帝传递了一个明确的信号："我知道你在乎什么，我能拿走你最在乎的东西。"这种"以人质为筹码"的策略，在政治斗争中屡见不鲜，但段延庆的使用方式——以一个普通人（钟万仇）的私人恩怨为掩护来实现自己的政治目的——更加狡猾和隐晦。

**虎啸与龙吟的结局**

黄眉僧与段延庆的对弈，以僵局收场。这种结局暗示了权力角逐的复杂性和无解——正邪双方势均力敌，谁也无法轻易取胜。虎啸与龙吟，最终没有分出胜负，而是形成了一种微妙的平衡。

这种"平局"的结局，在叙事上具有多重功能。首先，它保持了故事的悬念——读者不知道段延庆会不会成功夺回皇位，不知道段誉会不会被救出。其次，它为后续的情节发展留下了空间——如果黄眉僧赢了，段延庆就会失败，故事就会提前结束；如果段延庆赢了，段誉就会陷入绝境。只有"平局"，才能使故事继续发展下去。第三，它体现了金庸对"力量平衡"的理解——在这个江湖中，没有任何一方能够完全压倒另一方，力量永远是动态变化的。

### 六、小说中的"解救"模式——群体智慧与个体勇气

《天龙八部》中的"解救"情节，展现了金庸对"群体智慧"与"个体勇气"的思考。在段誉被囚万劫谷的事件中，金庸完整地展示了"解救"的全过程——从发现问题、分析问题、制定方案到执行方案，每一个环节都有不同的人物发挥作用。

在文学理论中，"解救"（rescue）是一种常见的叙事模式。从古希腊神话中的忒修斯解救阿里阿德涅，到中国古典小说中的"英雄救美"，这种模式在文学史上有着悠久的传统。金庸在《天龙八部》中对"解救"模式的运用，既继承了这一传统，又赋予了它新的内涵——他不是简单地写"英雄救美"或"兄弟相救"，而是通过"解救"这一事件，展现了一个团队的协作、智者的谋略和勇者的牺牲。

金庸的"解救"模式还有一个重要的特点：他往往会将"解救"的过程写得极为详细——从发现问题、分析问题、讨论方案、制定计划到执行计划，每一个环节都不省略。这种"详写"的方式，一方面增加了小说的真实感——读者会觉得自己仿佛也在参与营救，另一方面也展现了人物的性格——不同的人在面对危机时会有不同的反应和贡献。

**群体智慧的体现**

段誉被囚万劫谷后，保定帝不是一个人去救他，而是召集了段正淳、高升泰、巴天石、范骅、华赫艮等一众智囊，共同商议对策。华赫艮提出挖地道的方案，展现了"技术专家"在危机中的价值。黄眉僧的出场，则体现了"盟友"的重要性。

金庸通过这一情节，传达了"众志成城"的理念——一个人无法解决的问题，一群人合作就能找到办法。保定帝作为"领导者"，他的作用不是"想主意"，而是"选主意"——他召集了各个领域的专家，听取了各种不同的意见，最终选择了最适合的方案。

这种"集体决策"的模式，在中国传统文化中有着深厚的渊源。"三个臭皮匠，顶个诸葛亮"——这句俗语说的就是群体智慧的价值。金庸将这种传统文化中的智慧融入小说中，使"解救段誉"这一情节不仅仅是一个"英雄救英雄"的故事，更是一个"group problem solving"的案例。

**个体勇气的价值**

然而群体智慧并不能解决所有问题。在解救段誉的过程中，个体的勇气同样不可或缺。黄眉僧冒着生命危险与段延庆对决，华赫艮放下身段重操旧业，范骅和巴天石甘愿"做盗墓的小贼"——这些个体的勇气，是解救行动成功的关键。

华赫艮的"放下身段"，在个体勇气中具有特别的意义。他已经是司徒（三公之一），在朝中享有极高的地位和尊严。然而为了救段誉，他愿意重操旧业——那个他曾经引以为耻的"盗墓"行业。这种"为救人而放下尊严"的选择，是比"舍生取义"更加艰难的一种勇气——因为死是一时的痛苦，而"放下尊严"却是长期的牺牲。

**段誉的自我解救**

段誉在石室中并未坐以待毙。他以凌波微步行走来压制春药的药力，以棋力指点黄眉僧，以机智应对段延庆。他既是"被解救者"，也是"自救者"。这种"自救"意识，是段誉成长的重要标志——他不再是一个完全被动的人，而是开始主动参与自己的命运。

段誉的"自我解救"，在《天龙八部》中具有重要的象征意义。它表明段誉已经从"被动接受命运安排的人"变成了"主动参与命运创造的人"。在石室中，他没有哭泣哀嚎，没有等待别人来救他，而是利用自己仅有的资源——凌波微步、棋力、智慧——来帮助自己渡过难关。

### 七、原著第八章详解："虎啸龙吟"

以上六个小节从宏观层面分析了《天龙八部》的权力博弈与救赎之路。以下进入小说本身，逐章细读原著内容。

#### 7.1 章节标题

**"虎啸龙吟"**

这是《天龙八部》第八章的标题。"虎啸龙吟"指的是猛虎的咆哮和神龙的吟唱，在中国传统文化中，虎啸和龙吟分别象征着两种强大力量的展示。在本章中，"虎"指段延庆——他以"恶贯满盈"的姿态出现在万劫谷，如同下山猛虎；"龙"指保定帝和黄眉僧——他们代表了正统和正道的力量，如同天龙在渊。虎啸龙吟，暗喻正邪两种力量在本章中的激烈碰撞。

从标题的文学效果来看，"虎啸龙吟"四字气势磅礴，读来令人热血沸腾。它与第七章的"无计悔多情"形成了鲜明的对比——第七章阴柔悲伤，第八章阳刚雄浑。这种风格的交替变化，使金庸的小说始终保持着节奏感——一张一弛、一阴一阳，使读者不会感到单调。

#### 7.2 保定帝的营救部署

段誉和木婉清被囚万劫谷的消息传回大理城后，保定帝立即召集众人商议营救方案。刀白凤心急如焚，恳求皇上尽快救人。保定帝虽然也极为担心，但面上不动声色，以"让誉儿多经历一些艰难，磨练磨练"来安抚众人。

保定帝心中清楚，囚禁段誉的不是普通的江湖中人，而是四大恶人之首——"恶贯满盈"段延庆。此人武功深不可测，又占据地利，强行硬攻必然造成重大伤亡。他需要一个既能救人又不会激化冲突的方案。

保定帝的"不动声色"，在政治学上可以称为"危机管理中的镇定"。在危机中，领导者的情绪会直接影响下属的士气。如果保定帝表现出慌乱和焦虑，大理朝廷也会陷入恐慌。而他保持镇定，就给了下属一种"一切尽在掌握"的信心。这种领导艺术，在《天龙八部》的政治叙事中是一个重要的主题——金庸通过对保定帝、段正淳、高升泰等政治人物的描写，展现了一个"好皇帝"应该具备的素质和行为准则。

保定帝的营救部署，体现了他作为一国之君的"全局观"。他没有因为段誉是他的侄子就贸然行动，而是冷静地分析了形势，评估了各种可能的方案，最终选择了"以智斗代替武斗"的策略。这种"冷静决策"的能力，是领导者最宝贵的品质之一。金庸通过保定帝的形象，向读者展示了一个"理想的君主"应该具备的素质——仁厚但不软弱，冷静但不冷酷，果断但不莽撞。

#### 7.3 华赫艮的地道之计

大理国三公——司徒华赫艮、司马范骅、司空巴天石——被召入宫中商议。范骅思前想后，提出了一个大胆的方案——挖掘地道。

他笑道："小弟何敢取笑大哥？我是想咱们混进万劫谷中，挖掘一条地道，通入镇南世子的石室，然后神不知、鬼不觉地救他出来。"

华赫艮一拍大腿，叫道："妙极，妙极！"他于盗墓一事有天生嗜好，二十年来虽然再不干此营生，偶尔想起，仍是禁止不住手痒。华赫艮当即绘出万劫谷的地形图，拟订地道的入口路线。至于如何避人耳目、如何运出地道中所挖的泥土等等，原是他的无双绝技。

这是金庸笔下一个极富趣味的场景——三位高官放下身段，以"盗墓"的方式来救人。华赫艮的盗墓经验在此刻发挥了意想不到的作用。

华赫艮的"地形图"和他对地道的规划，是这一解救行动的关键。金庸虽然没有详细描写这些技术细节，但他通过华赫艮"沉吟半晌"和"绘出地形图"的动作，使读者感受到——华赫艮的专业知识在此时发挥了重要作用。这种"专业主义"的精神——即使是一个盗墓贼的专业知识，也能够在关键时刻发挥作用——是金庸想要传达的价值观之一。

#### 7.4 黄眉僧的出场

就在众人商议之际，保定帝请来了一位重要的帮手——**黄眉僧**。黄眉僧是一位得道高僧，与保定帝交情深厚。五年前，黄眉僧曾为大理通国百姓请命，求保定帝免了盐税，保定帝直到此时方允，双方心照不宣——那是务必替他救出段誉。

黄眉僧的出场方式极具气势——他以精湛的棋艺和深厚的内力，与段延庆展开了一场惊心动魄的对决。两人以青石为棋盘，黄眉僧用铁槌在石上刻画棋路，段延庆用铁杖在石上捺出印记。每一步棋都蕴含着内力，每一着都暗藏杀机。

黄眉僧以"哀兵必胜"的信念对上段延庆——他自知棋力不如对方，但他愿意为救段誉而拼上性命。这种"舍身取义"的精神，与段延庆的"恶贯满盈"形成了鲜明的对比。

黄眉僧与保定帝之间的"默契"——保定帝免了盐税，黄眉僧替他救段誉——体现了中国传统文化中的"报恩"思想。黄眉僧不是为了"正义"而出手，而是为了"报恩"——他五年前求保定帝免了盐税，保定帝当时没有答应，但一直记着这件事。此刻他请黄眉僧出手，黄眉僧自然义不容辞。这种"恩怨分明"的江湖规则，是金庸小说中常见的行为准则。

#### 7.5 旁观者清——段誉的棋局智慧

段誉自幼善弈，隔墙观棋，忍不住出声指点。他的每一步棋都直指要害，令黄眉僧茅塞顿开，棋路顺畅。段延庆虽然棋力高深，但在段誉的指点下，黄眉僧竟然逐渐扳回了劣势。

段誉虽然被囚禁在石室中，身中春药，性命垂危，却在棋局上保持清醒的判断力。这种"身处困境而心不困"的能力，正是段誉最可贵的品质。他以棋力为武器，间接参与了这场对决，展现了他的智慧和才华。

段誉的棋局智慧，在《天龙八部》的叙事中具有特殊的地位。它表明——段誉的价值不仅仅在于他是一个"会武功的王子"，更在于他是一个"有智慧的文人"。他的武功在大部分时候"时灵时不灵"，但他的智慧始终是"时时清醒"的。即使在最危险、最绝望的环境中，他也能保持冷静的思考能力。

#### 7.6 "阴阳和合散"——段誉的春药之困

段延庆给段誉和木婉清服下的并非普通的毒药，而是一种名为"阴阳和合散"的春药。这种药并不会致命，却会催发人的情欲，使人无法自制。

段誉发现服下的是春药而非毒药后，心中反而一喜——原来不是他自己的欲望，而是药力所致。他开始以凌波微步在斗室中快步行走，只须走得一两个圈子，心头便感清凉。木婉清却身发高热，神智迷糊，大半时刻都是昏昏沉沉的倚壁而睡。

段誉凭借凌波微步压制住了药力。他虽然不会主动运用武功，但凌波微步在无意中救了他一命——不仅让他能够在战斗中自保，也在这伦理困境中帮助他守住了道德的底线。

"阴阳和合散"这个名字，本身就具有强烈的讽刺意味。"阴阳和合"在道教中是指阴阳两种力量的和諧统一，是最高境界。而段延庆给段誉和木婉清服下的"阴阳和合散"，却是要摧毀两人的道德底线，让他们做出违背伦理的事情。这种"以神圣之名行邪恶之事"的讽刺，在金庸的小说中屡见不鲜。

#### 7.7 段延庆的真相——前朝太子

随着情节的推进，段延庆的真实身份逐渐浮出水面。高升泰向保定帝奏报："这青袍客自称'恶贯满盈'，臣听他说话的声音依稀相似，此刻想来，他……他好像便是延庆太子。"

保定帝大吃一惊——延庆太子是上明帝的儿子，原本是皇位的合法继承人。当年朝中政变，延庆太子不知所踪，人人都以为他已经被杀了。没想到事隔多年，他竟然以"四大恶人"之首的身份重现江湖。

保定帝当即表态："皇位本来是延庆太子的。当日只因找他不着，上明帝这才接位，后来又传位给我。延庆太子既然复出，我这皇位便该当还他。"

这一表态，既展现了保定帝的仁厚，也暗藏着危机——段延庆以段誉为筹码，要逼迫保定帝让位。救段誉一事，已经从"江湖恩怨"升级为"皇位之争"。

段延庆的"前朝太子"身份，在叙事上具有强烈的戏剧性。一个被宣称死亡的人，突然以"恶人"的身份复活——这种"死而复生"的设定，是金庸小说中常见的情节手法。但金庸没有将段延庆简单地塑造成一个"复仇者"，而是深入挖掘了他的悲剧——他不再是一个"想要夺回王位的反派"，而是一个"被命运彻底摧毁的人"。

#### 7.8 本章的功能与意义

"虎啸龙吟"作为整部小说的第八章，完成了以下叙事功能：

| 功能 | 说明 |
|------|------|
| **段延庆正式出场** | "四大恶人"之首亮相，其身份与大理国命运紧密相连 |
| **黄眉僧的棋局对决** | "文斗"的经典场景，棋力与内力的双重较量 |
| **段誉棋力展现** | 段誉以旁观者的身份指点棋局，展现智慧与才华 |
| **华赫艮的盗墓绝技** | 三公合作挖地道，幽默中展现大理智囊团的团结 |
| **"阴阳和合散"之困** | 段誉以凌波微步压制春药，道德与欲望的再次较量 |
| **保定帝的仁君形象** | 面对皇位被夺的危险，保定帝以退让展现胸怀 |
| **段延庆的身份揭露** | 延庆太子重现人间，为后续身世谜团埋下伏笔 |
| **万劫谷主线的推进** | 营救段誉的行动全面展开，多条线索汇合 |

金庸在本章中，以"虎啸龙吟"为核心意象，将段誉被囚万劫谷这一事件提升到了"皇位之争"的高度。段延庆的出场，不仅给段誉带来了巨大的危机，也为后续的身世真相埋下了最重要的伏笔。与此同时，黄眉僧与段延庆的棋局对决、华赫艮的地道之计，则展现了金庸在情节设计上的卓越才华——既有智谋的较量，又有武力的交锋，还有命运的博弈，三者交织在一起，使本章成为前八回中结构最为复杂、内容最为丰富的一章。

从本书的叙事结构来看，第八章在整部《天龙八部》中扮演着"承前启后"的关键角色。前七回以段誉的"奇遇"和"困境"为主线，展现了段誉从一个不谙世事的少年到一个经历风浪的青年的成长。从第八章开始，段誉的故事将进入一个更加复杂、更加广阔的叙事空间——万劫谷之争只是前奏，真正的大冲突将在后续章回中全面展开。第八章的"虎啸龙吟"，不仅是一场"正邪对决"的战斗，更是《天龙八部》宏大叙事的"序曲"——它预示着一场更大的"龙争虎斗"即将上演。

在段延庆与保定帝的"虎啸龙吟"之外，本章还揭示了《天龙八部》中一个更为深刻的主题：权力的虚妄。段延庆一生追求皇位，为此付出了身体残疾、灵魂堕落的代价。然而当他在万劫谷面对保定帝时，他发现自己的追求——那个皇位——对于保定帝来说，并不像他想象的那么重要。保定帝愿意为了段誉的安全而放弃皇位——这表明，在保定帝心中，亲情比权力更加重要。段延庆与保定帝的对比，揭示了权力本质上的"虚无"——那些用尽一生去追求权力的人，往往在得到权力的那一刻发现，权力并没有想象中那么有价值。
