# 天龙八部 · 深度解析网页版

> **无人不冤，有情皆孽**

金庸先生尝言："《天龙八部》这部书，主要写的是人间的苦难。"自1963年在《明报》连载以来，这部鸿篇巨制便以其恢弘的气魄、复杂的人性与深邃的佛学哲思，震动了整个华语文学界。

本网站是对《天龙八部》全书的深度解读，包含完整的五十回逐回解析、人物分析、佛学命题讨论，以及金庸叙事艺术的专题研究。

## 🛠️ 技术说明

本网站为纯静态 HTML 页面，无任何外部依赖，可直接在浏览器中打开或部署到任何静态托管服务。

### 文件结构

```
jingyong_tlbb/
├── index.html                          # 首页（本网站入口）
├── 天龙八部简介-排版版.html            # 全本纪事（排版版）
├── 深度解析目录.html                   # 50回深度解析目录
├── 天龙八部-第X章-深度解析.html        # 章节深度解析（1-10章）
├── 天龙八部-第XX回-深度解析.html       # 回目深度解析（11-50回）
├── 批量转换深度解析.py                 # MD→HTML转换脚本
├── 添加标点.py                         # 中文标点修复脚本
├── README.md                           # 本文件
└── .gitignore
```

### 生成工具

- 深度解析内容以 Markdown 编写，通过 `批量转换深度解析.py` 自动生成 HTML
- `添加标点.py` 用于为无标点文本自动添加中文标点
